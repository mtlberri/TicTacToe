## Getting Started


Clone the present repository on your local machine, and run "TicTacToe.py":

In your terminal: 
```
$ python TicTacToe.py
```

## Using

Press your num pads with numbers in between 1 and 9 to mark the board:
7|8|9
4|5|6
1|2|3

Press 'q' to quit.
